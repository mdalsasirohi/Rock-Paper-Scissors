# ✊📄✂️ Rock Paper Scissors

An interactive **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**. Players can choose Rock, Paper, or Scissors and compete against a computer-generated choice.

## ✨ Features

* Player vs Computer gameplay
* Random computer choices
* Interactive buttons
* Automatic winner detection
* Dynamic display of player and computer choices
* Score tracking
* Reset game functionality
* Simple and interactive user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

rock-paper-scissors/
│
├── index.html
├── style.css
└── script.js

## 🚀 How to Run

1. Clone this repository:

   ```bash
   git clone YOUR_REPOSITORY_URL
   ```
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Choose **Rock ✊**, **Paper 📄**, or **Scissors ✂️** and start playing!

## 🎯 Learning Outcome

Through this project, I practiced JavaScript concepts such as functions, conditional statements, random number generation, DOM manipulation, event handling, and dynamic score updates.
